## Tech Stack

- **React 19** + Vite
- **Tailwind CSS v4** - Mobile-first styling
- **React Router** - Client-side routing
- **Zustand** - State management
- **react-hook-form + Zod** - Form handling & validation
- **lucide-react** - Icons
